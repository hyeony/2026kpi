# H9 CUPPIN

**H9 CUPPIN은 팀원의 커피 취향을 기억하고, 반복되는 주문 과정을 간소화하기 위해 만든 AI 기반 커피 주문 도우미입니다.**

> *매번 메신저로 "뭐 드실래요?"를 묻고 주문을 취합하지 않아도, AI가 팀원의 취향을 기억하고 주문을 빠르게 정리해 줍니다.*

---

## 배경

티타임이 생길 때마다 메신저를 오가며 주문을 취합하고, 구성원의 취향을 다시 확인하고, 주문 내용을 정리하는 과정은 생각보다 꽤(?) 반복적이고 귀찮은 업무였습니다.

**"이 과정을 AI가 대신 기억하고 정리해 줄 수는 없을까?"**

H9 CUPPIN은 이러한 작은 불편함을 줄이고, 커피 주문에 쓰는 시간을 팀원 간의 소통과 협업의 시간으로 바꾸기 위해 시작된 프로젝트입니다.

---

---

## CUPPIN이 하는 일

CUPPIN은 커피 주문을 **세 가지 역할**로 나눠 처리합니다.

### 1. 취향 기억

팀원별로 선호 음료, 취향 태그, 주문 시 참고할 메모를 저장해 둡니다.  
한 번 등록하면 다음 주문부터 다시 묻지 않아도 됩니다.

### 2. 주문 구성

오늘 함께 시킬 사람만 골라 주문 목록을 만듭니다.

- AI 입력 *(예: "FE팀 주문 목록 만들어줘")*
- 조직도에서 팀·개인 선택
- 즐겨찾기 / 최근 주문 불러오기 *(프로토타입: 샘플)*

### 3. 주문 정리

선택된 인원의 취향을 바탕으로 **카페·단톡에 바로 붙일 수 있는 주문 문구**를 자동 생성합니다.

- 참여자별 음료 상세
- 음료별 집계
- AI 메모·취향 참고 사항

> AI는 "누구랑 커피 마실래?"를 연결해 주는 도구가 아니라, **주문 담당자의 취합·정리 업무**를 돕는 비서에 가깝습니다.

---

## 이렇게 씁니다

### Step 1. 팀원 취향 등록 *(최초 1회)*

구성원 탭에서 팀원을 확인하고, 각자(또는 담당자가) 선호 음료와 취향 메모를 등록합니다.

```
[스크린샷: 구성원 / 프로필 취향 설정]
```

### Step 2. 오늘 주문할 사람 선택

홈에서 AI로 팀을 불러오거나, 조직도에서 직접 선택합니다.  
필요하면 게스트를 1회성으로 추가할 수 있습니다.

```
[스크린샷: AI 입력 또는 조직도 선택]
```

### Step 3. 주문 문구 복사 → 전달

하단에서 인원·잔 수·음료 요약을 확인하고, **주문 문구 복사** 버튼으로 단톡·카페에 전달합니다.

```
[스크린샷: 주문 문구 / 복사 UI]
```

## Before / After


| Before           | After            |
| ---------------- | ---------------- |
| 매번 "뭐 드실래요?"를 돌림 | 취향이 이미 저장되어 있음   |
| 담당자가 손으로 정리·집계   | 주문 문구가 자동으로 완성됨  |
| "저번이랑 같아요" 반복 확인 | 프로필에서 바로 불러옴     |
| 디카페인·메모 누락       | AI 메모 블록으로 함께 전달 |
| 담당자마다 정리 방식이 다름  | 동일한 포맷으로 인수인계    |


**기대 효과:** 주문에 쓰이던 시간을 줄이고, 팀원 간 소통·협업에 쓸 시간을 늘립니다.
