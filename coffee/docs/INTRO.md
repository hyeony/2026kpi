# H9 CUPPIN ☕

## 배경

커피 주문도 결국 업무 중 반복되는 작은 일이라고 생각했습니다.

티타임만 되면 메신저를 오가며 주문을 취합하고, 취향을 다시 확인하고, 주문 내용을 정리하는 과정이 은근 반복되더라고요.

**"이 정도는 AI가 대신 기억해주면 좋겠다."**

그렇게 H9 CUPPIN을 만들어보게 됐습니다.

☕ **"뭐 드실래요?"는 조금 덜 하고, 주문은 조금 더 편하게.**



  
"CUPPIN"은 "Cup + In"에서 착안한 이름으로,  
커피 한 잔에 함께한다는 의미를 담았습니다.

---

# CUPPIN이 하는 일

CUPPIN은 커피 주문을 **세 단계로 조금 더 편하게** 만들어주는 도구입니다.

## 1. 취향 기억

팀원별 선호 음료와 취향, 주문 시 참고할 메모를 저장합니다.

한 번만 등록해두면 다음 주문부터 다시 물어볼 필요가 없습니다.

---

## 2. 오늘 주문할 사람 선택

오늘 함께 주문할 사람만 선택하면 됩니다.

- AI 입력 *(예: "FE팀 주문 만들어줘")*
- 조직도에서 직접 선택
- 즐겨찾기 / 최근 주문 불러오기 *(프로토타입 샘플)*

---

## 3. 주문 문구 자동 생성

선택한 팀원의 취향을 바탕으로 카페나 단체 채팅에 바로 붙여 넣을 수 있는 주문 문구를 만들어줍니다.

- 참여자별 주문 내역
- 음료별 수량 자동 집계
- 취향 메모 함께 반영

> AI가 주문을 대신하는 것이 아니라,
> **주문 담당자의 반복되는 취합과 정리를 조금 덜어주는 역할**을 합니다.

### 기대 효과

- 반복되는 주문 취합 시간을 줄일 수 있습니다.
- 담당자의 작은 번거로움을 덜어줍니다.
- 커피 주문을 조금 더 쉽고 빠르게 만들 수 있습니다.
